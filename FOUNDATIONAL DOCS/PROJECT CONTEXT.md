Project Context & Strategy: Vortix Vacuum 4-in-1
1. Objective
Goal: Validate the product, market fit, and marketing angle for the "Vortix Vacuum 4-in-1" via a dry test.
Methodology: E-commerce Advertorial Funnel.
Success Metric: Number of unique clicks on the final "Check Availability" / "Claim Offer" CTA on the Sales Page (Intent to purchase).
Current State: Dry testing mode. backend/checkout is NOT required yet.
2. Funnel Flow
Traffic Source (External)
Pre-Sales Page (Advertorial)
Purpose: Educate the user, agitate the problem (recirculating dust, allergies), and introduce the solution (Vortix).
Status: ~80% complete.
Key Action: User clicks CTA ("Claim 50% Off" / "Check Availability").
Sales Page (The Offer)
Purpose: Sell the OUTCOME (not just features/benefits). Make the user feel the result of owning the product.
Reference: "Sherum ErgoSeat" style (Clean, visual, outcome-focused).
Key Action: User clicks final CTA to purchase.
"Sold Out" Pop-up (The Capture)
Trigger: Click on Sales Page CTA.
Message: "Sorry, we are currently sold out due to high demand."
Action: Form to collect email/contact info to notify when back in stock.
Logic: This simulates a conversion while capturing leads for future retargeting.
3. Technical Requirements
Tracking: A "pixel-like" mechanism to register unique users who clicked the final CTA. This is crucial for data accuracy (validating the angle).
Routing: Seamless redirection from Advertorial -> Sales Page.
Pop-up: High-converting design for the "Sold Out" modal on the Sales Page.
4. Product: Vortix Vacuum 4-in-1
Core Value Prop: Stops breathing recirculated dust.
Features (to translate to Outcomes):
9,500 PA Suction -> Deep clean, no leftover dirt.
HEPA Filtration -> Cleaner air, allergy relief.
4-in-1 -> Versatility (Home, Car, Sofa, Cracks).
5. Design Guidelines
Aesthetic: "Surreal Realism", Premium, High-End.
Sales Page Approach: Focus on the Life Transformation (e.g., "A specific outcome like 'Wake up without sneezing'").
